# Dotfiles for a minimal & feature rich terminal setup

This repository contains my dotfiles for a minimal and feature-rich linux desktop setup. It includes configurations for various tools and applications.

## Overview

My config is based on CachyOS with Hyprland, but most of the configurations should be adaptable to other distributions with tiling window managers.

There are lots of optimizations for convertible laptops, such as screenrotation, lid aware monitor configuration and stylus.

### Desktop Environment

- **Window Manager**: hyprland
- **Monitor Configuration**: hyprdynamicmonitors
- **Status Bar**: waybar
- **Wallpaper Manager**: waypaper / wofi / hyprpapr
- **Lock Screen**: hyprlock / matugen
- **Power menu**: wofi
- **Idle Manager**: hypridle
- **Blue light filter**: hyprsunset
- **No Login Manager**

### Terminal and Shell

- **Terminal Emulator**: kitty / alacritty
- **Shell**: zsh
- **Shell Prompt**: powerlevel10k
- **Shell Tools**: fzf, zoxide, eza
- **Zsh plugin manager**: zinit
- **Zsh Plugins**:
  - zsh-autosuggestions
  - zsh-syntax-highlighting
  - zsh-completions
  - fzf-tab
  - OMZ:
    - archlinux
    - aliases
    - alias-finder
    - git
    - man
    - sudo

### Applications

- **File Manager**: yazi
- **Text Editor**: neovim
- **Screenshot Tool**: grimblast / alacritty

### Extras

- **Image Viewer**: feh
- **US Keyboard Layout with german Umlauts**
- **Services for Whatsapp and Signal Clients**
- **ncdu**
